\# 🚀 Portfolio — Pedro Lima



Bem-vindo ao repositório do meu Portfólio de Desenvolvedor Full-Stack.



Este projeto foi criado para apresentar minhas habilidades, experiências e projetos utilizando um layout moderno inspirado em landing pages de tecnologia.



Uma das principais características do projeto é o uso do Supabase para armazenar um arquivo `config.json`, permitindo atualizar o conteúdo do portfólio sem precisar alterar o código ou fazer novo deploy.



---



\# 🧠 Conceito do Projeto



O portfólio utiliza um arquivo de configuração (`config.json`) armazenado no Supabase.



Isso permite alterar facilmente informações como



\- Nome

\- Descrição

\- Links sociais

\- Projetos

\- Tecnologias

\- Contato

\- Experiências



Assim, qualquer mudança pode ser feita diretamente no banco de dados.



✅ Sem precisar alterar código  

✅ Sem precisar fazer novo deploy
